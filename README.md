# student-scores
Python scripts for student score processing
Pythonで学生情報を管理し、合計・平均・ランク・フィルター機能を備えたプロジェクトです。
## Features / 機能
- Load student data from CSV / CSVから学生データを読み込む
- Calculate total, average, and rank / 合計・平均・ランクを計算
- Export processed CSV / 処理結果をCSVに出力
- Filter students by average / 平均点で学生をフィルター
## Technologies / 使用技術
- Python 3.x
- Object-Oriented Programming (OOP)
- CSV module
